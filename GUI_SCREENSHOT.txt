ОСНОВНЫЕ ЭЛЕМЕНТЫ ИНТЕРФЕЙСА:
==============================

1. ПОЛЕ ПОИСКА
   - Ввод поискового запроса
   - Поддержка Enter для быстрого поиска
   - Плейсхолдер с подсказкой

2. НАСТРОЙКИ ПОИСКА
   - Количество результатов (1-100)
   - Минимальная релевантность (0.0-1.0)
   - Адаптивные значения по умолчанию

3. КНОПКИ УПРАВЛЕНИЯ
   - Поиск: выполнение поиска
   - Очистить: сброс результатов
   - Добавить документ: новый документ
   - Удалить документ: удаление выбранного
   - Перестроить индекс: обновление индекса

4. РЕЗУЛЬТАТЫ ПОИСКА
   - Заголовок документа
   - Балл релевантности (4 знака после запятой)
   - Совпавшие ключевые слова
   - Двойной клик для просмотра фрагмента

5. СТАТИСТИКА СИСТЕМЫ
   - Общее количество документов
   - Уникальные слова в индексе
   - Среднее количество слов в документе
   - Общий размер в символах

6. СТРОКА СОСТОЯНИЯ
   - Текущий статус операции
   - Прогресс-бар для длительных операций
   - Информация о количестве результатов
